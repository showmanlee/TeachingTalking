GameManager.cards = new List<List<CardStatus>>();
cards[0][0] = 1-1번 카드

class CardStatus:
	카드 이름 - string
	카드군 - int
	카드 번호 - int
	카드 결합형 - List<CardCombine>
	
class CardCombine:
	결합명 - string
	결합 위치 - bool
	결합되는 카드군 - int
	가중치 - int
	문장 종료 - bool


카드의 결합: 카드1 - 카드2 - 카드3
한 쪽으로만 붙는 카드(1, 3)는 그 카드만 반영.
양쪽으로 붙는 카드(2)는 앞에 붙는 모양과 뒤에 붙는 모양을 모두 고려해 표시할 것.
이 때 최대한 특수 형태가 발현되는 곳을 따라가거나, 양쪽 모두 특수 형태가 발현된다면 그 둘을 합칠 수 있도록 할 것(가중치에 따라).
가중치는 같은 형태의 결합이 등장할 때마다 +1하는 수치

학습의 경우
단어 학습에서는 입력한 이름을 카드 이름에 저장하기(끝)
구문 학습에서는 카드 조합을 한다
	카드 조합 구조에서, 카드의 결합 구조에 맞게 카드 결합을 저장하고, 가중치를 추가한다.
	그리고 카드 조합식과 그 결과를 따로 저장해둔다. 실제로 테스트할 때 참고할 수 있도록.
문장 학습에서는 카드 조합을 하는데, 구분 학습에서 나온 카드 조합(저장했던)을 고려한다.
